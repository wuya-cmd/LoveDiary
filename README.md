# 情侣日记微信小程序

这是一个基于微信小程序开发的情侣日记应用，参考了开源项目BearDiary的设计理念，专门用于记录和保存情侣相处的美好时光。

## 功能特点

- 📝 **文字记录**：支持添加文字描述，记录心情和感受
- 📸 **照片保存**：支持上传多张照片，最多可添加9张
- 📅 **时间排序**：日记按时间倒序排列，最新的显示在最前面
- 💾 **本地存储**：数据保存在本地，保护隐私安全
- 🎨 **美观界面**：采用粉色系设计，界面温馨浪漫
- 📱 **响应式设计**：适配不同屏幕尺寸的设备

## 主要功能

### 1. 日记列表页（index）
- 显示所有日记的列表
- 按时间倒序排列
- 显示日记的日期、时间、内容摘要和图片预览
- 支持下拉刷新
- 空状态提示

### 2. 添加日记页（add）
- 文字输入框（最多500字）
- 图片上传功能（最多9张）
- 图片预览和删除
- 字符计数显示
- 保存按钮

### 3. 日记详情页（detail）
- 显示完整的日记内容
- 大图显示所有照片
- 图片预览功能
- 编辑和删除按钮

## 技术特点

- 使用微信小程序原生开发
- 采用本地存储（wx.setStorageSync/wx.getStorageSync）
- 响应式布局设计
- 渐变色背景和卡片式布局
- 平滑的动画效果

## 使用说明

1. 在微信开发者工具中导入项目
2. 确保开发者工具版本支持基础库2.0以上
3. 在模拟器或真机中预览效果
4. 点击"添加日记"按钮创建新的日记
5. 输入文字内容和上传照片
6. 保存后可在列表页查看所有日记
7. 点击日记可查看详情

## 注意事项

- 数据存储在本地，清除小程序数据会丢失所有日记
- 图片使用临时路径，建议定期备份重要照片
- 编辑功能待开发，目前只能删除日记
- 建议在真机测试以获得最佳体验
