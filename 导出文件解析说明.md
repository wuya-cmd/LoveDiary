# LoveDiary 导出文件解析说明书

## 一、文件格式概述

LoveDiary Android 应用支持两种导出格式：

| 格式 | 文件名模式 | 存储位置 |
|------|-----------|---------|
| JSON备份 | `diary_backup_YYYYMMDD_HHmmss.json` | Downloads/LoveDiary/ |
| 小程序.bak | `diary_YYYYMMDD_NNN.bak` | 由小程序生成 |

---

## 二、Android JSON 备份格式

### 2.1 文件结构

```json
{
  "version": "1.2",
  "exportTime": "2024-01-15T10:30:00.000Z",
  "totalCount": 10,
  "anniversary_date": "2020-01-01",
  "diaries": [ ... ]
}
```

### 2.2 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `version` | String | 导出格式版本，当前为 "1.2" |
| `exportTime` | String | ISO格式导出时间 |
| `totalCount` | Integer | 日记总数 |
| `anniversary_date` | String | 相识日期（可选） |
| `diaries` | Array | 日记数组 |

### 2.3 日记对象结构

```json
{
  "id": "diary_xxx",
  "content": "日记正文内容",
  "createTime": "2024-01-15T10:30:00.000Z",
  "updateTime": "2024-01-15T10:30:00.000Z",
  "privacyLevel": 0,
  "category": "默认分类",
  "tags": "标签1,标签2",
  "images": [ ... ]
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | String | 日记唯一标识 |
| `content` | String | 日记正文（JSON转义） |
| `createTime` | String | 创建时间 ISO格式 |
| `updateTime` | String | 更新时间 ISO格式 |
| `privacyLevel` | Integer | 0=公开, 1=私密 |
| `category` | String | 分类名称 |
| `tags` | String | 标签，逗号分隔 |
| `images` | Array | 图片数组 |

### 2.4 图片对象结构

```json
{
  "fileName": "IMG_20240115.jpg",
  "originalPath": "/storage/.../original.jpg",
  "compressed": true,
  "compressionQuality": 85,
  "originalSize": 2048000,
  "compressedSize": 512000,
  "format": "JPEG",
  "base64Data": "/9j/4AAQSkZJRgABAQEAeAB4AAD..."
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `fileName` | String | 文件名 |
| `originalPath` | String | 原始路径（可能为空） |
| `compressed` | Boolean | 是否经过压缩 |
| `compressionQuality` | Integer | 压缩质量 0-100 |
| `originalSize` | Long | 原始大小（字节） |
| `compressedSize` | Long | 压缩后大小（字节） |
| `format` | String | 图片格式(JPEG/PNG/WebP) |
| `base64Data` | String | 图片Base64数据（NO_WRAP） |

## 三、小程序 .bak 格式

小程序导出的 `.bak` 文件本质是 **JSON格式**，每篇日记单独保存为一个文件。

### 3.1 文件结构

```json
{
  "id": "diary_xxx",
  "content": "日记正文内容",
  "createTime": "2024-01-15T10:30:00.000Z",
  "updateTime": "2024-01-15T10:30:00.000Z",
  "privacyLevel": 0,
  "images": [ ... ]
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | String | 日记唯一标识 |
| `content` | String | 日记正文 |
| `createTime` | String | 创建时间 ISO格式 |
| `updateTime` | String | 更新时间 ISO格式 |
| `privacyLevel` | Integer | 0=公开, 1=私密 |
| `images` | Array | 图片数组 |

### 3.2 小程序图片对象结构

```json
{
  "fileName": "image_123.jpg",
  "base64Data": "/9j/4AAQSkZJRgABAQEAeAB4AAD...",
  "originalPath": "/http://tmp/...",
  "originalSize": 2048000,
  "base64Size": 1536000,
  "compressed": true,
  "compressionType": "smart",
  "finalSize": 512000,
  "finalSizeMB": "0.49"
}
```

## 四、隐私等级定义

| 值 | 常量名 | 说明 |
|----|--------|------|
| 0 | `PUBLIC` | 公开 - 所有人都可以查看 |
| 1 | `PRIVATE` | 私密 - 需要验证身份后查看 |

## 五、数据恢复示例

### 5.1 Python 解析示例

```python
import json
import base64

def parse_diary_backup(json_file):
    """解析Android导出的JSON备份文件"""
    with open(json_file, 'r', encoding='utf-8') as f:
        data = json.load(f)
    
    print(f"版本: {data['version']}")
    print(f"导出时间: {data['exportTime']}")
    print(f"日记数量: {data['totalCount']}")
    print(f"相识日期: {data.get('anniversary_date', '未设置')}")
    
    for diary in data['diaries']:
        print(f"\n--- 日记 {diary['id']} ---")
        print(f"内容: {diary['content'][:100]}...")
        print(f"创建时间: {diary['createTime']}")
        print(f"隐私等级: {'私密' if diary['privacyLevel'] == 1 else '公开'}")
        print(f"分类: {diary['category']}")
        print(f"标签: {diary['tags']}")
        
        # 恢复图片
        for img in diary.get('images', []):
            img_data = base64.b64decode(img['base64Data'])
            with open(img['fileName'], 'wb') as imgf:
                imgf.write(img_data)
            print(f"已恢复图片: {img['fileName']}")

def parse_miniprogram_bak(bak_file):
    """解析小程序导出的.bak文件"""
    with open(bak_file, 'r', encoding='utf-8') as f:
        diary = json.load(f)
    
    print(f"日记ID: {diary['id']}")
    print(f"内容: {diary['content']}")
    print(f"创建时间: {diary['createTime']}")
    
    for img in diary.get('images', []):
        if img.get('base64Data') and not img.get('error'):
            img_data = base64.b64decode(img['base64Data'])
            with open(img['fileName'], 'wb') as imgf:
                imgf.write(img_data)
            print(f"已恢复图片: {img['fileName']}")
```

### 5.2 JavaScript 解析示例

```javascript
const fs = require('fs');

function parseDiaryBackup(filePath) {
    const data = JSON.parse(fs.readFileSync(filePath, 'utf8'));
    
    console.log(`版本: ${data.version}`);
    console.log(`日记数量: ${data.totalCount}`);
    
    data.diaries.forEach(diary => {
        console.log(`\n--- 日记 ${diary.id} ---`);
        console.log(`内容: ${diary.content.substring(0, 100)}...`);
        console.log(`隐私等级: ${diary.privacyLevel === 1 ? '私密' : '公开'}`);
        
        diary.images?.forEach(img => {
            if (img.base64Data && !img.error) {
                const buffer = Buffer.from(img.base64Data, 'base64');
                fs.writeFileSync(img.fileName, buffer);
                console.log(`已恢复图片: ${img.fileName}`);
            }
        });
    });
}
```

## 六、注意事项

1. **Base64编码**: Android导出的Base64使用 `Base64.NO_WRAP` 标志，解析时需注意
2. **特殊字符转义**: JSON中的内容已转义（`\n`, `\r`, `\"`等）
3. **时间格式**: 均为ISO 8601格式：`yyyy-MM-dd'T'HH:mm:ss.SSS'Z'`
4. **图片大小限制**: 超过10MB的图片不会导出
5. **字符编码**: 文件使用UTF-8编码
